安装pip2

pip2 也只能使用官方脚本 get-pip.py 安装。记得把脚本第一行的 python 改成 python2，因为上一步，我们只创建了 python2 的连接，没有创建 python 的连接。主要考虑到，你的机器上可能已经存在了 python 连接，并指向了 python3
